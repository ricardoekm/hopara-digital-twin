# Template Service

Node.js + TypeScript. Manages visualization templates.

## Main commands

```bash
yarn start              # Development mode (nodemon, port 8089)
yarn build              # Compile TypeScript to out/
yarn test               # Tests
yarn lint               # Lint
```

## Environment variables

| Variable       | Description            |
|----------------|------------------------|
| `DATASET_URL`  | Dataset service URL    |
| `RESOURCE_URL` | Resource service URL   |

## Structure

```
src/            # TypeScript source code
templates/      # Visualization templates
out/            # Compiled output — do not edit
```

## Conventions

- TypeScript strict
- `out/` directory is generated by the build — do not edit
