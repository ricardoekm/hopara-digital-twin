# Notification Service

Node.js + TypeScript. Notification service (port 8085).

## Main commands

```bash
yarn start              # Development mode (nodemon)
yarn build              # Compile TypeScript to out/
yarn test               # Tests
yarn lint               # Lint
```

## Structure

```
src/            # TypeScript source code
out/            # Compiled output — do not edit
```

## Conventions

- TypeScript strict
- `out/` directory is generated by the build — do not edit
