# BFF Service

Node.js + TypeScript. Gateway between the frontend and internal services.

## Main commands

```bash
yarn start              # Development mode (nodemon, port 8086)
yarn build              # Compile TypeScript to out/
yarn test               # Tests
yarn lint               # Lint
yarn pre-commit         # lint + test (run before committing)
```

## Environment variables

| Variable           | Description                      |
|--------------------|----------------------------------|
| `DATASET_URL`      | Dataset service URL              |
| `VISUALIZATION_URL`| Visualization service URL        |
| `TENANT_URL`       | Tenant service URL               |
| `TEMPLATE_URL`     | Template service URL             |
| `RESOURCE_URL`     | Resource API URL                 |

## Structure

```
src/            # TypeScript source code
out/            # Compiled output — do not edit
```

## Conventions

- DI with awilix
- TypeScript strict
- `out/` is generated by the build — do not edit
